4.2.5 商户信息管理

本节主要介绍商户信息的添加、修改与删除。

商户信息添加：

1.  进入【信息管理】 -- 【商户信息管理】页面；

2.  点击【添加】按钮，录入所有信息；

3.  点击【确定】按钮，保存商户信息；

4.  保存成功后，商户信息会自动添加到下面的列表中；

商户信息编辑：

1.  进入【信息管理】 -- 【商户信息管理】页面；

2.  点击要编辑的商户信息对应的【编辑】按钮；

3.  点击后该条消息会填充到相应的文本框中，对信息进行修改，修改后点击【确定】按钮；

4.  保存成功后，商户信息会自动添加到下面的列表中；

商户信息删除：

1.  进入【信息管理】 -- 【商户信息管理】页面；

2.  点击要删除的商户信息对应的【删除】按钮；

3.  删除成功后，被删除的商户信息将在下面的列表中删除。

图10 商户信息管理添加页面

图10说明：

1.  添加操作：点击【添加】按钮，弹出新增配置页面；

2.  商场名称：通过下拉框选择（商场名称由商场管理模块提供），例如U6食堂、王府井；

3.  楼层：通过下拉框选择（楼层由地图管理模块提供）；如1F；

4.  选择定位点坐标：用户在地图上定位的点分别到y轴和x轴的距离，单位为米。（x轴和y轴由添加地图时所定的坐标原点决定）。只能由数字组成，最大长度为10位。定位点坐标值：（1）手动输入；（2）通过后台计算得出：点击【选择】按钮，弹出选择定位点坐标页面，在地图上选取坐标点，点击【确定】按钮得到定位点坐标的值；

图11 确定商户位置

5.  商家介绍：用户所定的定位点的商户的具体内容，在手机端点击定位点的图标会弹出商户的详细信息。可以由中文、字母、数字、特殊符号组成，最大长度为100位；

6.  VIP商户：是/否，用于客户端的商户信息是否在手机App启动时需要在页面上显示；

7.  选择图片（非必填项）：在服务器端选择相应的图片上传，图片格式支持：.jpg和.png；

8.  选择视频（非必填项）：在服务器端选择相应的视频上传，视频格式支持：.mp4；

9.  是否开启：选择是否要开启在地图上显示本条消息。选择开启后，手机端能接收到商户信息，选择关闭后，则不能接收到商户信息；

10. 确定操作：点击【确定】按钮，进行商户信息的添加；

11. 取消操作：点击【取消】按钮，取消对已添加的商户信息进行保存。取消添加后，列表中无新增商户信息；

12. 编辑操作：点击【编辑】按钮后，在输入部分展示该条信息供用户修改，用户修改后点击【确定】按钮，修改的数据信息将会覆盖原有的数据信息；

13. 删除操作：点击【删除】按钮，确认删除后，则该条数据将从列表中和数据库中删除，并刷新页面展示最新的消息信息；

14. 预览操作：点击【预览】按钮，弹出预览页面，可以预览图片和播放视频。
